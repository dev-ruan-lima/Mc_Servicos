# 🌐 Mc Serviços

Este projeto consiste no desenvolvimento de um **site institucional**, criado com foco em design moderno, clareza das informações e uma boa experiência do usuário.

O site tem como objetivo apresentar a empresa Mc serviços de forma profissional, destacando seus serviços, valores e canais de contato, com um layout responsivo e navegação intuitiva.

---

## 🎯 Objetivo do Projeto

Desenvolver um site institucional front-end que:
- Transmita credibilidade e profissionalismo
- Seja responsivo e acessível
- Possua uma estrutura clara e organizada

---

## ✨ Funcionalidades

- Layout moderno e clean  
- Design responsivo (desktop, tablet e mobile)  
- Navegação intuitiva  
- Seções informativas (sobre, serviços, contato, etc.)  
- Estrutura semântica em HTML  

---

## 🚀 Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**

---

## 📱 Responsividade

O site foi desenvolvido seguindo boas práticas de responsividade, garantindo uma experiência consistente em diferentes tamanhos de tela e dispositivos.
